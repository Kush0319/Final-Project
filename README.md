# 📈 Stock Market Analysis using Python

## 📖 Overview

This project analyzes historical stock market data using **Python**, **Pandas**, **NumPy**, **Matplotlib**, and **Seaborn**. It performs Exploratory Data Analysis (EDA) to identify market trends, stock performance, trading volume, and price movements through insightful visualizations.

---

## 🎯 Objectives

- 📊 Clean and preprocess stock market data
- 📈 Analyze stock price trends
- 📉 Calculate moving averages and daily returns
- 📦 Analyze trading volume
- 🔥 Visualize data using charts and graphs
- 📌 Generate meaningful insights from historical data

---

## 🛠️ Technologies Used

- 🐍 Python
- 🐼 Pandas
- 🔢 NumPy
- 📊 Matplotlib
- 🎨 Seaborn

---

## 📊 Features

- ✅ Data Cleaning
- ✅ Exploratory Data Analysis (EDA)
- ✅ Closing Price Analysis
- ✅ Trading Volume Analysis
- ✅ Moving Average Calculation
- ✅ Daily Return Analysis
- ✅ Correlation Heatmap
- ✅ Multiple Data Visualizations

---

## 📂 Dataset

The project uses historical stock market data containing:

- 📅 Date
- 💰 Open Price
- 📈 High Price
- 📉 Low Price
- 💵 Close Price
- 📊 Adjusted Close Price
- 📦 Trading Volume

---

## 🚀 Getting Started

Install the required libraries:

```bash
pip install pandas numpy matplotlib seaborn
```

Run the project:

```bash
python stock_analysis.py
```
